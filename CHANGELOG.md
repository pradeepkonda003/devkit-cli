# Changelog

## [0.1.0] - 2026-06-23

### Added
- `devkit validate` command for OpenAPI 3.x schema validation (JSON and YAML)
- `devkit generate` command for typed SDK stub generation in Python and TypeScript
- Strict mode (`--strict`) to fail CI on warnings
- JSON output format (`--format json`) for pipeline integration
- Dry run mode (`--dry-run`) to preview generated files
- GitHub Actions CI workflow with lint and test matrix (Python 3.11 + 3.12)
- 21 tests with 96% coverage
